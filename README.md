# Notepad-application
A notepad application with online Wikipedia search developed using Tkinter Library

# Requirements
 - Tkinter
 - wikipedia
 - beautifulsoup (bs4)

## Installation
```bash
$ pip install -r requirements.txt # bs4 wikipedia
```

# Keyboard shortcuts
| Keybind    | Effect                  |
|------------|-------------------------|
| `Ctrl + c` | Copy text               |
| `Ctrl + v` | Paste text              |
| `Ctrl + x` | Cut text                |
| `Ctrl + s` | Save file               |
| `Ctrl + o` | Open file               |
| `Ctrl + n` | New file                |
| `Ctrl + w` | Search on wikipedia     |
| `Ctrl + b` | Change background color |
| `Ctrl + f` | Change font color       |
